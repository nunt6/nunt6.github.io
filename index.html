linux基础

2022-12-19

linux系统与常用的windows系统不同，在linux系统下大多数操作都是依靠终端来进行的，就需要我们输入一些命令来告诉系统我们要做什么。下面我来总结一些我在学习过程中常用的一些命令。
1.转到指定文件夹：
cd+目标文件夹的绝对路径
在linux系统中很多操作都要在我们目标文件夹中进行所以使用这个命令能够抵达我们目标文件夹执行下一步操作。
2.查看当前文件夹的绝对路径：
pwd
在许多地方我们需要用到绝对路径（指文件在对于根目录真正存在的路径），使用这个命令
3.查看当前文件夹中的文件：
ls
通常这个命令会配合着其他参数一起使用，目前我最常使用al这两个参数让系统同时显示文件夹中所有文件以及文件的详情信息。
4.创建（删除）文件：
touch(rm)+文件名
当删除文件的命令加上-f的参数时系统会强制删除指定文件所以慎重使用这条命令。
5.创建（删除）文件夹：
mkdir(rmdir)+文件夹名
这条命令与创建文件的命令使用方法类似
6.复制（移动，重命名）文件（文件夹）：
cp(mv)+文件的绝对路径
在使用ssh远程连接时也用到了这条命令
7.显示文本内容：
cat+绝对路径
8.对文件提权：
chmod +? 文件名
？所代表的的是r/w/x，分别代表可读/可写/可执行,同时也可以用数字代替，记住r=4，w=2，x=1，可以用数字的和同时提多种权限，例如chmod +7 xxxx表示给xxxx文件同时加上了三个权限
9.使用管理员权限（root用户）
sudo +命令
如果要登录管理员用户则使用
sudo su
输入密码后就可以使用管理员权限。
注：在上文中提及的绝对路径，如果目标文件在当前目录下，可以直接使用文件名或者文件夹名，不一定使用绝对路径

记录一次使用ssh免密远程连接过程

2022-12-18

利用windows power shell实现在物理机和虚拟linux系统下的免密登录。
使用以下的命令生成公私钥对。
ssh-keygen
这个命令会自动在计算机.ssh目录下中自动生成公私钥。
id_rsa （私钥）
id_rsa.pub (公钥)
有了公钥只要将公钥上传到目的系统的.ssh目录下。
ssh-copy-id -i ~/.ssh/id_rsa.pub 目标用户名@目标ip
使用以下命令进行测试。
ssh 目标用户名@目标ip
没有输入密码就登录成功说明成功实现免密登录。
ps：在上传公钥时应该要输入密码。
linux系统IP地址查看可使用
ifconfig
目前已知的出现的问题：
1.在进行远程连接时未安装ssh服务
使用apt install ssh进行安装ssh服务
必要时在命令前面加上sudo提权。
2.ssh服务未开启,使用以下命令开启ssh服务
ssh service start
